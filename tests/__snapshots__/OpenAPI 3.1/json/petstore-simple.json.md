# Simple Petstore

This is a slimmed down single path version of the Petstore definition.

**Version:** 1.0.0

## Servers

- **URL:** https://httpbin.org

## pet

### Update a pet

This operation will update a pet in the database.

```http
PUT /pet/{id}
```

### Find a pet

This operation will find a pet in the database.

```http
GET /pet/{id}
```
